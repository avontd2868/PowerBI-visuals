Class TouchManager

This class "listens" to the TouchEventInterpreter to recieve touch events and sends it to all "Touch Delegates" with TouchRegions that contain the mouse event. Prior to sending off the event, its position is put in respect to the delegate's TouchRegion and converted to the appropriate unit (see IPixelToItem).
